SCREENSHOTS FOLDER
==================
Take 3 screenshots from your phone (or Chrome DevTools mobile view):

  screen-dashboard.png   ← Dashboard page  (1080 x 1920 px)
  screen-sales.png       ← Sales page       (1080 x 1920 px)
  screen-gst.png         ← GST Invoice page (1080 x 1920 px)

These are required for your Google Play Store listing.

TIP: Use Chrome DevTools → Toggle device toolbar → Set to 1080x1920
     Then screenshot each page.
